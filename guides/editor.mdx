The CodeAura editor is the starting point for every code image you create. Paste or type any snippet and the editor immediately applies syntax highlighting — no configuration required. From there, a set of focused settings let you shape the code's presentation before you reach the design panels.

## Entering your code

Open CodeAura and type or paste your snippet directly into the editor canvas. The editor loads with a TypeScript example (`fibonacci.ts`) to give you a sense of the output. Select all and paste to replace it with your own code, or click **Clear** to start from a blank state.

<Note>
The editor uses CodeMirror under the hood. Standard keyboard shortcuts for your OS — cut, copy, paste, undo, redo, select-all — all work as expected.
</Note>

## Language selection and detection

## Editor settings

These settings control how your code is rendered inside the editor. They are found in the editor toolbar above the code area.

### Tab size

Sets the number of spaces used to represent a tab character. The default is **2**. Changing this value re-renders existing indentation in the editor without modifying the underlying code.

### Word wrap

When off (the default), long lines extend horizontally and a scrollbar appears. Toggle **Word wrap** on to break long lines at the editor boundary — useful for wide prose or long strings that you want fully visible in a fixed-width image.

### Show whitespace

Renders visible markers for spaces and tab characters. Useful when you want to verify indentation or demonstrate whitespace-sensitive syntax (e.g. Python, YAML, Makefile) in your image.

### Show indent guides

Draws faint vertical lines at each indentation level. Indent guides help viewers follow nested code structure, particularly in deeply indented blocks.

### File name

The file name field (default: `untitled`) appears in the window title bar when **Show title** is enabled on the Frame panel. Changing it to a real file name — such as `api.ts` or `main.py` — adds authenticity to the image.

## Line count

CodeAura displays a live line count below the editor. This updates as you type and gives you a quick read on whether your snippet fits comfortably in the chosen frame width.
