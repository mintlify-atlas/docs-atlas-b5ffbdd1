CodeAura is a web-based tool that transforms your code snippets into polished, shareable images. Open the editor in your browser, paste any code, pick a theme, tweak the frame and font settings, then export a high-resolution PNG, SVG, or WebP — or share a link so others can view or remix your snapshot. No installation, no design skills, and no account required to get started.

## Who is CodeAura for?

CodeAura is built for developers who share code outside of a code editor — in a tweet, a blog post, a slide deck, a pull request comment, or a README. If you have ever taken a screenshot of your terminal and wished it looked better, CodeAura is the tool for you.

## How CodeAura works

CodeAura runs entirely in the browser. Visit the app, and you land straight in the editor — no sign-up gate, no download, no setup. The editor stores your work in local history automatically, so your last session is always there when you return. Sign in to sync history across devices and generate persistent shareable links.

## Supported languages

CodeAura supports over 50 languages. Languages marked **full** have active CodeMirror syntax highlighting; languages marked **display** are recognized for labeling and auto-detection but render in plain text mode.

### Full syntax highlighting

JavaScript, TypeScript, Python, Rust, Java, C++, C, HTML, CSS, JSON, Markdown, SQL, XML, PHP

### Display only

Go, Swift, Kotlin, C#, Ruby, Dart, Elixir, Haskell, Lua, Perl, R, Scala, Shell/Bash, PowerShell, YAML, TOML, GraphQL, SCSS, LESS, Dockerfile, Terraform, JSX, TSX, Vue SFC, Svelte, Solidity, Zig, Nim, Clojure, F#, OCaml, Erlang, LaTeX, Makefile, Nginx, Diff, INI, Regex, Assembly (x86), GLSL/WGSL, Nix, Gleam
